CSS

  • CSS defines the colors, aesthetics, visual layout, and organization.
  • Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
  • Internal Stylesheets are placed directly in a webpage's HTML file using a "style" element in the "head" element.
  • The most common way to style a webpage is by using an external CSS style sheet.
  • A "link" element in the head of the HTML document creates the connection to an external file.
  • A rel attribute specifies the relationship between the current document and the linked document resource. An href attribute specifies the location (URL) of the external resource.
  • Use the "*" selector to apply a rule to all the elements visible on the page.
  • Combine two selectors into a single CSS rule with a comma.
  • The "img" selector assigns property values to any image element in the markup.
  • The display property with the block value assigns each image to start on a new line.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git checkout main: switches back to the main branch

  • git config user.email: to know the user email
